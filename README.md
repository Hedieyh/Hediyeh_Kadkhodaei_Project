# 📱 Language Learning App (Flutter)

This project is a mobile user interface developed using Flutter for the course **COM206 - Visual Programming**.

---

## 🚀 Project Overview

This application is designed as a language learning platform with a clean, modern, and user-friendly interface.  
It includes multiple screens and focuses on smooth navigation and visual consistency.

---

## 🎯 Features

- Welcome Screen  
- Login Screen  
- Sign Up Screen  
- Goal Selection Screen  
- Clean and modern UI  
- Custom color theme  
- Smooth navigation between screens  

---

## 🧠 Technologies Used

- Flutter  
- Dart  
- Figma (for UI design)

---

## 🔄 Application Flow

The user starts from the welcome screen, then chooses to either log in or sign up.  
After authentication, the user proceeds to the goal selection screen.
